# PortFolio

Ce projet est un site web portfolio développé en **HTML, CSS, JavaScript et PHP**, permettant de présenter mes projets, compétences et expériences.

## Technologies utilisées

- **HTML5** et **CSS3**
- **JavaScript** (vanilla)
- **PHP** (>= 8.1)
- Base de données : phpmyadmin
